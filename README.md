
  # Algofin.ai

  This is a code bundle for Algofin.ai. The original project is available at https://www.figma.com/design/xvlbGAQmZ9OSyjVTf4AFit/Algofin.ai.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Deployment wiring

  See `DEPLOYMENT_SETUP.md` to connect GitHub, Vercel, Render, and Supabase so pushes from this repo publish updates automatically.
